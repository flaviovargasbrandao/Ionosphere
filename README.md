# Social Impact Tracker for Bands

This project aims to collect and analyze public social media activity from bands and correlate it with Spotify stream counts. The ultimate goal is to identify patterns and enable predictive analysis on the impact of posts over time.

## Project Goals

- Collect social media post data (Instagram, Twitter/X, YouTube, etc.)
- Retrieve Spotify play and popularity metrics
- Analyze temporal correlation between posts and stream increases
- Build dashboards to visualize performance
- Support future predictive models (AI-based)

## Tech Stack

- Python
- Spotipy – for Spotify API integration
- Graph APIs – for social data (Meta, Twitter, YouTube)
- Pandas, Plotly, Streamlit – data analysis and dashboards
- Git + Trello – version control and task tracking
